\# Minecraft 1.8.9 Pack Converter



A tool for converting old Minecraft 1.8.9 resource packs so they are more usable on newer Minecraft versions.



\## What it does

\- Renames old texture folders

\- Renames many old item and block texture filenames

\- Fixes some old GUI issues that break modern versions

\- Creates a converted pack zip automatically



\## Notes

This tool is best-effort. Some packs may still have missing or imperfect textures.



\## How to use

1\. Open the app

2\. Choose your old resource pack zip

3\. Click Convert

4\. Get the converted pack from the output folder
